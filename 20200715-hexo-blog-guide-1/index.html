  • hexo clean 用于清除缓存与静态文件。
  • hexo new "文章标题" 用于新建文章。
  • hexo server 用于启动本地服务器查看渲染好的网页，可简写为hexo s。
  • hexo generate 用于生成静态文件，可简写为hexo g。
  • hexo deploy 用于部署网站，需要先设置好_config.yml，可简写为hexo d。
